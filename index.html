Setup

You'll need to install certain programs for this class. Some must be installed, while others are only recommended. If you reset your computer, you are expected to reinstall the programs you need for the class.

Software

  1. Install the standard student software pack if you haven't already. Make sure that Google Drive is set up on your laptop so that your files sync to your computer.
  2. Install a good text editor like Notepad++ for Windows or TextWrangler for Mac. If you're looking for a challenge, try installing Vim. It's tough to learn at first, but it's very powerful.
  3. Install Git. You can use the default settings.
  4. Install KeePass so you don't lose passwords to important accounts. If you can't access an account or program because you forgot your password, you will receive a zero for the assignment.
  5. Install Firefox Developer Edition and Blisk. We will use these for web development.

Stretch Projects

Whenever you finish working on the assigned lessons, you are expected to go back and complete any optional assignments. Once those are complete, you are free to work on your stretch projects during class time. However all students must earn a certain number of points per quarter through stretch projects. Points earned beyond the required amount are extra credit.
