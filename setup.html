Setup

You'll need to install certain programs for this class. Some must be installed, while others are only recommended. If you reset your computer, you are expected to reinstall the programs you need for the class.

Software

Checkpoint 1: Student Software Pack

Install the standard student software pack if you haven't already. Make sure that Google Drive is set up on your laptop so that your files sync to your computer.

Checkpoint 2: Text Editor

Install a good text editor like Notepad++ for Windows or TextWrangler for Mac. If you're looking for a challenge,
try installing Vim. It's tough to learn at first, but it's very powerful. If you already have a text editor (not a word processor) that you like to use, then you may do so as long as it can handle what you need to do for class. Checkpoint 4: KeePass

Install KeePass so you don't lose passwords to important accounts. If you can't access an account or program because you forgot your password, you will receive a zero for the assignment.

Once you install KeePass, set it up with a master password. Write down the master password on a piece of notebook paper. You will need to store this later for homework.

Accounts

Checkpoint 6: Github

Go to Github and create an account. You must use your school email address. If you have a personal Github account, you can use it but you must change your account email to your school email address or create a new account using your school email address.

Once you've created an account and logged in, open Git GUI on your computer and click on Help in the menu. Choose Show SSH Key. If you already have a key, copy it to your clipboard. If you don't, click Generate. You may leave the password blank if you prefer. Then copy it to your clipboard.


Go to your Github profile and choose SSH and GPG Keys from the list on the left side. Then click New SSH Key and choose a title for the key. Then paste the key into the Key section and click Add SSH Key.
