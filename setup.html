Checkpoint 3: Git

Install Git. You can use the default settings.

If you are using Windows, go to the Start Menu and search for Git Bash. Open it and type in the following commands, replacing the example email with your school email and the example name with your name. If you are using a Mac, open the Terminal and run it from there.

git config --global user.email "johndoe@seoulchristian.org"

git config --global user.name "John Doe"

Close Git bash and install Git LFS. You can use the default settings. Then open Git Bash and type the following command.

git lfs install

Now if you need Git LFS for an assignment, you can use it.

Checkpoint 4: KeePass

Install KeePass so you don't lose passwords to important accounts. If you're using a Mac, look for KeePassX. If you can't access an account or program because you forgot your password, you will receive a zero for the assignment.

Once you install KeePass, create a new KeePass database and set it up with a master password. Write down the master password on a piece of notebook paper. You will need to store this later for homework.

Accounts

Checkpoint 6: Github

Go to Github and create an account. You must use your school email address. If you have a personal Github account, you can use it but you must change your account email to your school email address or create a new account using your school email address.

Save your Github password in KeePass.

Once you've created an account and logged in, open Git GUI (Windows only) on your computer and click on Help in the menu. Choose Show SSH Key. If you already have a key, copy it to your clipboard. If you don't, click Generate. You may leave the password blank if you prefer. Then copy it to your clipboard.


If you're using macOS, follow these instructions to check if you have an SSH key already. If you don't, then create an SSH key. If you do, then follow these instructions to add your SSH key to Github.

Go to your Github profile and choose SSH and GPG Keys from the list on the left side. Then click New SSH Key and choose a title for the key. Then paste the key into the Key section and click Add SSH Key.
