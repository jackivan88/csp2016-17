Taking Screenshots

You'll have to take screenshots of your work throughout this course, so it's important to learn how to do it quickly and correctly. If you're using macOS, then you have some tools built in that we'll use. If you're using Windows, you need to install ShareX since the built-in Windows screenshot tools aren't quite good enough for what we need to do. Go to the section for your operating system and complete the checkpoints.

  1. macOS
  2. Windows

macOS

Before you do anything else, read this article published by Apple that explains basic screenshot keyboard shortcuts.

Take a screenshot of your entire screen. Find the file on your desktop and rename it takingScreenshotsCheckpoint1.png and attach it to the Taking Screenshots assignment on Google Classroom.
