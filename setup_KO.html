체크포인트 3: Git

Git 을 설치하십시오. 기본설정을 사용하십시오.

윈도우 사용자라면, 시작메뉴를 여신후 Git Bash를 찾으십시오. Git Bash를 여신후 다음과 같은 명령을 입력하십시오. 예시의 이메일주소와 이름은 당신의 이메일과 이름으로 바꿔넣으십시오. 맥 사용자라면, Terminal을 여신후 명령을 입력하십시오.

git config --global user.email "johndoe@seoulchristian.org"

git config --global user.name "John Doe"

Git bash를 닫으신후 Git을 설치하십시오. LFS. 기본 설정을 사용하셔도 좋습니다. Git Bash를 여신뒤 다음과 같이 입력하십시오.

git lfs install

지금부터 과제를 위해 Git LFS 를 사용하실수있습니다.

Mr. Olinda께 체크포인트를 검사받기위해 손을 드십시오.

체크포인트 4: KeePass

체크포인트 6: Github

Github으로 가신후 계정을 만드십시오. 반드시 학교 이메일 계정을 사용하셔야합니다. 만약 개인 Github 계정이 있으시다면, 그것을 사용하셔도 좋지만 이메일 계정을 반드시 학교 이메일 계정으로 바꾸시거나 새로 만드십시오.

Github 비밀번호를 keepass에 저장하십시오.

계정을 만드시고 로그인까지 하셨다면, Git GUI (윈도우 전용) 를 연후에 메뉴에 Help 를 누르십시오. Show SSH Key를 누르십시오. 이미 key가 있다면, 클립보드에 복사하십시오. 없으시다면, Generate를 누르십시오. 원한다면 비밀번호칸은 비워두셔도 졸습니다. 그후에 클랍보드로 복사하십시오.


맥 사용자라면, 에 들어가서 이미 SSH key가 있는지 확인하기위해 안내를 따르십시오.. 만약 없다면, 로 들어가서 SSH key를 만드십시오. 만약 있으시다면, 로 가셔서 안내를 따른뒤 SSH key를 Github에 추가하십시오..

Github profile에 가신후 왼쪽 면에서 SSH and GPG Keys 를 고르십시오. 그다음 New SSH Key 를 누르신뒤 이름을 입력하십시오. 그후 key를 Key section에 붙여넣으신뒤 Add SSH Key를 누르십시오.

Mr. Olinda께 체크포인트를 검사받기위해 손을 드십시오.
