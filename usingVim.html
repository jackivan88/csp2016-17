Using Vim

Vim is one of the text editors that's installed on many Unix-based systems. It is mode-based, which seems unusual at first.

Checkpoint 1: VIM Adventures

The first thing you should do is try out a game called VIM Adventures. This introduces basic Vim commands, and especially teaches you to use the h, j, k, and l keys for navigating in Vim. Keep playing until you finish Level 3 before moving on to the next tutorial. You can complete the game later if you want.

Checkpoint 3: Installing Vim

Close all open programs other than Chrome. Download and install the latest version of Vim for your operating system from their website. Mac users can also install Vim using Homebrew. Then launch it. On Windows it is called gVim, and on macOS it is called MacVim, unless you installed it using Homebrew.

Checkpoint 4: Vim Tutor

Now you're going to use Vim to learn about Vim. By now, you should be very familiar with the basic commands, and this will cover them again. But this time it will be in the actual Vim program. In Windows you can launch it by searching for "Vim tutor" and on macOS you can launch it by typing "vimtutor" in your terminal. Finish the entire tutorial.
